# MSME-Guard 🚨
### Predict. Prevent. Protect.

MSME-Guard is a predictive maintenance system for MSMEs that detects machine failures early and helps owners take immediate action — preventing costly breakdowns.

---

## 📌 Problem

Machines don't fail suddenly —  
**owners find out too late.**

- No early fault visibility  
- Failures discovered hours later  
- No immediate action  

💰 Results in **₹10,000 – ₹50,000 loss per breakdown**

---

## 💡 Solution

MSME-Guard converts early machine signals into **clear decisions and actions**.

- Detects faults before breakdown  
- Shows cost impact (₹1,200 vs ₹47,000)  
- Alerts owner instantly  
- Enables action (call + mechanic dispatch)

---

## ⚙️ How It Works

Sensors → Device → Processing → Decision → Action

- Machine data is captured using sensors  
- System detects early anomalies  
- Converts into a cost-based decision  
- Owner is notified and action is triggered

## 🔧 Tech Stack

- ESP32 + Sensors (IoT hardware layer)
- scikit-learn — Isolation Forest, Random Forest (ML models)
- FastAPI — ML inference backend
- CrewAI — multi-agent fault analysis & advisory
- React + Vite + Tailwind CSS (Frontend)
- VAPI — AI-powered voice call integration (owner alert)
